Security Analyses

Security analyses are essential when planning the development of the transmission system. Most transmission grid codes define compliance with the N‑1 security criterion as mandatory. In addition to N‑1 analysis, investigation of the broader N‑X security criterion is occasionally carried out.

N-1 Security Analysis

This type of security analysis encompasses investigation of N-1 security criterion which means power flows are calculated in many cases, defined by the user, where single branch, line or transformer, is out of service. Most often, power flows are calculated for the unavailability of each power grid branch recording overloads of available branches and violations of voltage limits at particular nodes.
