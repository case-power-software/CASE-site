Network Reduction

Network reduction is relatively straightforward function based on the power flow results, i.e. power flow calculation has to be carried out before the function is used. The user selects particular area(s) or all of them and certain voltage levels within the selected areas that are to be eliminated. CASE deletes all nodes, lines, and transformers that are connected to selected voltage levels in selected areas and introduces loads equal to power flows through the eliminated transformers. It should be noted that, after network reduction, the power flows in the remaining grid elements remain unchanged. On the other side, grid losses are reduced.
