Asynchronous Motor Start

This simulation model illustrates direct-on-line (DOL) start of the asynchronous motor (AM) the rated power and voltage of which are 2 MW and 10 kV, respectively. Single line diagram of the model is given in the next figure. As illustrated in the figure, the AM is initially disconnected from the grid, and a start event is defined to initiate its connection. The event connects AM to the grid at t=0.3 s.

CASE Implementation

Calculation of open conductor faults has been implemented in CASE as static calculation, like short circuits. CASE supports calculation of open conductor faults on lines, transformers, generators, and loads. By clicking "Open Conductors" button in the ribbon, the dialog window for calculation of open conductor faults is lunched. There are four tabs in the dialog, each for one type of power system element. When lines are selected, the distance between the first and second line node must be specified as a percentage (in this case, 40%), as illustrated in the next figure. This indicates that the parameters of the line from the first node to the fault correspond to 40% of the total line parameters.

Fig. 2. Launching open conductor faults module.

Since all three cases for open conductor fault are selected for calculation (1 phase open (PO) = TRUE, 2 PO = TRUE, 3 PO = TRUE), clicking button "Calculate" results of calculation for all three cases are printed in "Convergence and Results" window, as shown in the next figure. Conductor has two broken ends at the fault location, denoted as p and p' in the Fig. 1. CASE outputs the following at the fault location: voltages between the broken ends (Vpp′a(b)(c)), phase voltages on each side of the broken line (Vpa(b)(c) and Vp′a(b)(c)), and line voltages on each side of the broken line (Vpab(c) and Vp′ab(c)).
