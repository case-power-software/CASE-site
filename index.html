Welcome to CASE website hosted on GitHub. CASE is power system analysis software that performs almost all of power system calculations (see: About CASE) with interactive single line diagram implemented as object-oriented user interface. Compared to commercial software of the same type, CASE has original, easy to use, solutions regarding design and user interface (see: Application Design).
