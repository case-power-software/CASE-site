Welcome to CASE website hosted on GitHub. CASE is power system analysis software that performs almost all of power system calculations (see: About CASE) with interactive single line diagram implemented as object-oriented user interface. Compared to commercial software of the same type, CASE has original, easy to use, solutions regarding design and user interface (see: Application Design). CASE provides the following types of power system calculations:

Power Flow & Related Functions

  • Power (Load) Flow - AC and DC
  • N-1 and N-X Security Analysis
  • Neto Transfer Capacity (NTC)
  • PTDF and OTDF Calculation
  • Network Reduction
  • EnergyPLAN Scenarios Analysis

Faults

  • Short Circuits
  • Open Conductor Faults

Stability

  • Transient Stability
  • Small-Signal Stability
  • Voltage Stability

Unique Application Features

  • Graphical display of all power system overloads and voltage limit violations in a single view, regardless of system size.
